# Milestone v1.0: milestone

**Status:** ✅ SHIPPED 2026-05-19
**Phases:** 1-3
**Total Plans:** 3

## Overview

Initial release of the personal CV site — complete redesign with dark/neutral professional theme, single-column classic resume layout, About page with bio/skills/education, accessibility improvements, and visual polish.

## Phases

### Phase 1: Visual Foundation & Homepage Redesign

**Goal:** Apply the dark/neutral professional theme across the site and redesign the homepage as a single-column classic resume layout.
**Depends on:** None
**Plans:** 1 plan

Plans:

- [x] 01-01: Global theme, header, footer, homepage redesign, and index page composition

**Details:**
- Dark/neutral color scheme (`#0f0f0f` bg, `#e8e8e8` text, `#4a90d9` accent)
- System font stack via Tailwind `@theme` (serif headings, sans body)
- Single-column centered layout with `max-w-[720px]` container
- Header with serif "GG" logo, nav links, and social icons (LinkedIn, GitHub)
- Footer with dark surface background, muted text, no Supabase credit
- Homepage with name, title, bio, chronological experience entries, social links
- Valid HTML nesting (fixed `<li>` inside `<li>` bug)
