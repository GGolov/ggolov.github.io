# Phase 1 — UI Design Contract

> Visual and interaction contract for Phase 1: Visual Foundation & Homepage Redesign.
> Generated by gsd-ui-researcher (inline mode — agents not installed), verified manually.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | Tailwind CSS v4 (utility classes via `@import "tailwindcss"` in `global.css`) |
| Preset | not applicable |
| Component library | none — raw Astro components with Tailwind utility classes |
| Icon library | SVG icons (existing `linkedin-icon.svg`, `github-icon.svg` in `src/assets/`) |
| Font | System font stack — no web font loading. Use Tailwind's `font-sans` and `font-serif` utilities. |

### Font Stack Details

| Usage | Stack |
|-------|-------|
| Body text | System sans-serif (`font-sans`): `ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji"` |
| Headings | Serif (`font-serif`): `ui-serif, Georgia, Cambria, "Times New Roman", Times, serif` |
| Monospace (if needed) | System monospace (`font-mono`): `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace` |

> **Rationale:** System fonts eliminate loading delay, reduce requests, and render instantly. Using serif for headings and sans-serif for body creates a classic resume typographic hierarchy without external dependencies.

## Spacing Scale

Declared values (all multiples of 4, mapped to Tailwind defaults):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (`p-1`) | Icon gaps, inline padding |
| sm | 8px (`p-2`) | Compact element spacing, list item gaps |
| md | 16px (`p-4`) | Default element spacing, card padding |
| lg | 24px (`p-6`) | Section padding, header/footer padding |
| xl | 32px (`p-8`) | Layout gaps between major sections |
| 2xl | 48px (`p-12`) | Major section breaks, page margins |
| 3xl | 64px (`p-16`) | Page-level spacing, hero padding |

Exceptions: none

## Color

### Theme: Dark / Neutral with Cool Blue Accent

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0f0f0f` (near-black) | Page background, main surfaces |
| Secondary (30%) | `#1a1a2e` (dark navy gray) | Cards, sidebar panels, section backgrounds |
| Surface | `#16213e` (dark blue-gray) | Elevated surfaces, headers, footers |
| Text Primary | `#e8e8e8` (off-white) | Body text, primary content |
| Text Secondary | `#a0a0b0` (muted gray) | Secondary text, dates, metadata |
| Accent (10%) | `#4a90d9` (cool blue) | Links, interactive elements, emphasis |
| Accent Hover | `#6bb3f0` (light blue) | Hover states for links and buttons |
| Border / Divider | `#2a2a3e` (subtle gray-blue) | Section dividers, borders, hr |
| Destructive | `#e74c3c` (red) | Destructive actions only (not used in v1) |

### Tailwind Mapping

| Semantic Role | Tailwind Classes | Notes |
|---------------|-----------------|-------|
| Page bg | `bg-[#0f0f0f]` or custom color in `global.css` | Main background |
| Surface bg | `bg-[#1a1a2e]` | Section cards |
| Text | `text-[#e8e8e8]` | Primary text |
| Muted text | `text-[#a0a0b0]` | Metadata |
| Links | `text-[#4a90d9] hover:text-[#6bb3f0]` | Interactive elements |
| Dividers | `border-[#2a2a3e]` | Section separators |

Accent reserved for: links, navigation active state, social link icons on hover, emphasis highlights.

### Footer Layout
- Dark navy gray (`#16213e`) background
- Right-aligned: "Powered by Astro and hosted on GitHub Pages"
- Small text (`text-sm`, `#a0a0b0`)
- No Supabase credit
